# Getarise Blog

Welcome to the Getarise Blog! This is a simple blog built with Next.js.

## Getting Started

To get started with this project, clone the repository and install the dependencies:

```bash
git clone https://github.com/getarise/getarise.github.io.git
cd getarise.github.io
npm install
```

To run the development server:

```bash
npm run dev
```

This will start the server at `http://localhost:3000`.

## Building for Production

To build the project for production, run:

```bash
npm run build
npm start
```

## Features

- Home page with hero section and featured articles.
- Blog listing page.
- About page.
- Contact page.
